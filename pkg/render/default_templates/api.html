API Reference

Base URL

{{.BaseURL}}/api/v1

GET /schemas

Returns a JWS compact serialization containing all registered attestation schemas.

curl {{.BaseURL}}/api/v1/schemas.jwt

The payload contains a data array of SchemaMeta objects.

GET /schemas/{schemaId}

Returns a JWS compact serialization containing a single SchemaMeta object.

curl {{.BaseURL}}/api/v1/schemas/{uuid}.jwt

Signature Verification

Public keys are published as a JWKS at {{.BaseURL}}/api/v1/.well-known/jwks.json.
